STEPS ON HOW TO CREATE A GITHUB PAGE
[1] Create a Public Repository
[2] Have an index.html file
[3] In the settings, find the Pages tab.
[4] Under the Build and deployment section, Source must be "Deploy from a branch" and Branch must be "main" and "/(root)"
[5] Refresh the Site
[6] Visit the site.
